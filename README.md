# CSE310-module3-Unity-Flarpy_Borb

This is a unity game that mimics the game flappy bird.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download "Flarpy Borb Build.zip" from the github.
2. Extract the file.
3. Run the "Flarpy Borb" application inside of the file. (Note that there isn't a way to close the game inside the game so you may have to use something like alt + f4).

Instructions for using the software:

1. Press space to flap.
2. Try to get through as many pipes as possible.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Download Unity 6.4
* Download Visual Studio to work with Unity
